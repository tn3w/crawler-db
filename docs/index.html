How it works.

Crawlerdex unifies Crawlerdex patterns with live block-rate telemetry from robots-radar, which scans the top 25 000 websites' robots.txt daily.

1 · Search

Type any token: gpt, ahrefs, bing, to filter crawlers by name, description, or instance string.

2 · Paste a UA

Drop a full User-Agent header into the same box. The longest-matching pattern wins; the panel reveals which crawler it is.

3 · Filter by category

Tap a chip: ai-crawler, seo, scanner, to narrow by tag.

4 · Read the graph

Each crawler's modal shows the % of top-25k sites blocking it over time. Higher line = more rejected.
